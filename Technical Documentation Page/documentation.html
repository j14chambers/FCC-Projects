How to Make a Circle

A circle is pretty simple to make. At some point, I was trying to make one and I could not get to round evenly.

A circle is made by making the border-radius half the width and height. Just adjust the height and width according to your perferance.

 <p id="circle"><p>
#circle { 
    width: 200px; 
    height: 200px;
    background: #e0777d;;
    -moz-border-radius: 100px; 
    -webkit-border-radius: 100px; 
    border-radius: 100px;
}

How to make a Parallelogram

I was wondering how to get a square to slat on one side. So I looked at a parallelogram. I don't know much about skewing but it's used in making the sides slat.

By using skew, you can change the shape easily. Each transform is for different browsers.

 <p id="parallelogram"><p>
#parallelogram { 
    width: 150px; 
    height: 100px;
    -webkit-transform: skew(20deg);
    -moz-transform: skew(20deg);
    -o-transform: skew(20deg);
    background: #e0777d; 
}

How to Center a Button

Yep, I forgot how to center a button. It is a simple concept.

Put the button inside a div and use text-align to center the button.

<div class="btn-div">
    <button>Button A<button>
</div>
            
button {
    font-size: 18px;
    color: white;
    margin: 15px;
    padding: 25px 60px;
    background: #56667a;
    border: 5px solid gold;
}
.btn-div{
    text-align: center;
}

Adding Shadowing to a Button

box-shadow is used to adjust the shadows around a object.

 <button> Button 1 <button>
button {
    font-size: 18px;
    margin: 15px;
    padding: 20px 60px;
    background: #56667a;
    border: 3px solid gold;
    box-shadow: 0px 10px 20px;
}

How to Change the Color of the Progress Bar

I did not know that this tag exisited. When I saw it, I added to a random site that I was working on.

70%

So think of max as 100%. It can be any value, acting as the total number of items. The value is how much is completed or left, depending on how you do it.

 <progress id="progress-bar" max="100" value="70"><progress>
progress{
    border: 2px solid #9993a3;
    background: #e0777d;
    color: #56667a;
    margin: 40px 0 80px;
    height: 50px;
    width: 60%;
}
