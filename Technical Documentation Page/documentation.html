How to Make a Circle

A circle is made by making the border-radius half the width and height.

 <p id="circle"><p>
#circle { 
    width: 200px; 
    height: 200px;
    background: #e0777d;;
    -moz-border-radius: 100px; 
    -webkit-border-radius: 100px; 
    border-radius: 100px;
}

How to make a Parallelogram

 <p id="parallelogram"><p>
#parallelogram { 
    width: 150px; 
    height: 100px;
    -webkit-transform: skew(20deg);
    -moz-transform: skew(20deg);
    -o-transform: skew(20deg);
    background: #e0777d; 
}
